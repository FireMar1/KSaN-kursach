## HLS Streaming Service (Курсовая работа)

Веб-сервис на Python/FastAPI для загрузки видео, конвертации в HLS и воспроизведения в браузере.

### Возможности

- Клиент-серверная архитектура: браузер (UI + JS) и FastAPI-сервер.
- Загрузка медиафайлов на сервер.
- Фоновая конвертация видео в HLS (`.m3u8` + `.ts`) через `ffmpeg`.
- Хранение метаданных в SQLite (таблица `videos`).
- Администрирование: добавление и удаление видео только по admin-токену.
- Авторизация администратора через логин/пароль (`admin/admin123` по умолчанию).
- Роли пользователей в БД (`admin`, `viewer`) и журнал действий (`audit_logs`).
- Автообновление списка видео и статусов на клиенте (каждые 5 секунд).
- Базовая защита: проверка расширения, ограничение размера файла, проверка авторизации.

### Быстрый запуск

1. Установить зависимости:
   - `python3 -m venv .venv`
   - `source .venv/bin/activate`
   - `pip install -r requirements.txt`
2. Убедиться, что установлен `ffmpeg`:
   - `ffmpeg -version`
3. (Опционально) задать данные администратора:
   - `export ADMIN_USERNAME="admin"`
   - `export ADMIN_PASSWORD="admin123"`
4. Запустить сервер:
   - `uvicorn app.main:app --host 127.0.0.1 --port 8001 --reload`
5. Открыть:
   - [http://127.0.0.1:8001](http://127.0.0.1:8001)

### Админ-режим в UI

В верхней части страницы выполните вход как администратор.
После входа доступны:
- загрузка видео,
- удаление видео,
- просмотр журнала действий.

### Архитектура (для курсовой)

- **Data layer:** `database.py`, `models.py` (SQLite, таблицы `videos`, `users`, `audit_logs`).
- **Business logic:** `services.py` (конвертация в HLS), `auth.py` (авторизация/роли/сессии).
- **Presentation layer:** `routers.py` (API + HTML), `templates/index.html`, `static/main.js`.
- **Сетевое взаимодействие:** клиент делает HTTP-запросы к FastAPI (`/upload`, `/delete`, `/videos`, `/admin/*`), сервер отдает HLS по `/hls/...`.
